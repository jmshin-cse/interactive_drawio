Draw.io XML to Graph

Connected Path Highlighter

Draw.io XML을 붙여넣거나 업로드하면 노드/엣지 그래프로 파싱하고, 클릭한 노드를 기준으로 연결된 모든 영역을 한 번에 강조합니다.

입력

XML / mxGraphModel

현재 하이라이트 방식은 방향성을 무시한 연결 컴포넌트 탐색입니다. 즉, 클릭한 노드와 간접적으로라도 연결된 모든 노드와 엣지를 함께 강조합니다.

요약

로컬 입력
Pages 0
Nodes 0
Edges 0
Warnings 0

선택 노드

BFS Highlight
선택 없음 노드를 클릭하면 연결된 전체 영역을 강조하고 통계를 보여줍니다.

파싱 로그

Status
샘플을 불러오거나 XML을 붙여넣은 뒤 렌더링하세요.

작동 방식

mxCell[vertex="1"]는 노드, mxCell[edge="1"]는 엣지로 읽습니다. 그런 다음 인접 리스트를 만들고, 클릭한 노드에서 양방향 BFS를 수행해서 연결 컴포넌트 전체를 수집합니다.

선택/연결 짙은 파란색과 굵은 선으로 강조
비연결 영역 낮은 opacity로 흐리게 처리
기본 상태 전체 다이어그램을 원래 색상으로 렌더링

Ready

Draw.io XML을 그래프로 변환해 보세요

기본 좌표가 있으면 Draw.io 배치를 최대한 유지하고, 좌표가 불충분하면 Cytoscape 기본 레이아웃으로 자동 배치합니다.

연결 컴포넌트 하이라이트 노드를 클릭하면 해당 노드와 연결된 모든 노드/엣지를 강조하고, 나머지는 흐리게 처리합니다.
