Draw.io XML to Graph

Connected Path Highlighter

Draw.io XML을 붙여넣거나 업로드하면 노드/엣지 그래프로 파싱하고, 클릭한 노드를 기준으로 상류/하류 경로만 분리해서 강조합니다.

입력

XML / mxGraphModel

현재 하이라이트 방식은 선택 노드 기준 상류/하류 경로 탐색입니다. 즉, 선택 노드로 들어오는 경로와 선택 노드에서 나가는 경로만 따라가며 강조합니다.

요약

로컬 입력
Pages 0
Nodes 0
Edges 0
Warnings 0

선택 노드

Upstream / Downstream
선택 없음 노드를 클릭하면 상류/하류 경로만 강조하고 통계를 보여줍니다.

파싱 로그

Status
샘플을 불러오거나 XML을 붙여넣은 뒤 렌더링하세요.

작동 방식

mxCell[vertex="1"]는 노드, mxCell[edge="1"]는 엣지로 읽습니다. 그런 다음 방향성 인접 리스트를 만들고, 클릭한 노드에서 incoming BFS + outgoing BFS를 각각 수행해 상류/하류 경로만 수집합니다.

선택/연결 짙은 파란색과 굵은 선으로 강조
비연결 영역 낮은 opacity로 흐리게 처리
기본 상태 전체 다이어그램을 원래 색상으로 렌더링

Ready

Draw.io XML을 그래프로 변환해 보세요

Draw.io 원본 모델을 직접 렌더링해서 위치, 도형, 텍스트, 선 색을 최대한 원본 그대로 유지합니다.

경로 하이라이트 노드를 클릭하면 해당 노드의 상류/하류 경로만 강조하고, 나머지는 흐리게 처리합니다.
